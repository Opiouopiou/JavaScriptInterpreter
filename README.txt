This is a tool to help modding the game FoC and will provide the following functionality:

Change folder:
 - Navigate to a folder that contains the imagemeta.js with correct syntax. Select this file and click "Open"
 - Looks at imagemeta.js and then searches folder for images with the same names as meta has saved

Click on an image in the grid:
 - will display the image on the right display screen and will load any data attached in the corresponding imagemeta.js
 - change the data as you like tied to this image and click "update image data" underneath to save your changes
 - click on "update image data" to save your changes

Update folder:
 - ! remove any excess metafiles that have no corresponding image (#.jpg) THIS CANNOT BE UNDONE
 - ! convert any png, webp or JPG to jpg
 - rename any jpg image with unconventional name to unique "#.jpg"
 - add any images in the folder which are not yet in metafile to metafile and to gui's grid
 - any new data created with this tool will be filled with "unknown" in all fields

Update folder and all subfolders: 
 - same functionality as "Update folder" but it will also run in all subfolders of the chosen folder
